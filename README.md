== Cara menggunakan modul ==

1. Pertama, copy path di mana folder petra058_modul berada. Sebagai contoh, pada laptop saya folder tersebut terletak pada bagian Download.
![Tutorial](https://github.com/petrabona/Project-Ujian-Akhir-Semester-Struktur-Data-PSTI-A/blob/d5f738db4ee8534fb77e8caf338bac62a396baef/gambar/Screenshot%202026-06-08%20000745.png)

2. Setelah itu, buka Terminal (atau cmd) dan tuliskan prompt berikut: 'cd (path yang telah dicopy'. Sebagai contoh, saya memasukkan 'cd C:\Users\ADVAN\Downloads' pada Terminal.
![Tutorial](https://github.com/petrabona/Project-Ujian-Akhir-Semester-Struktur-Data-PSTI-A/blob/d5f738db4ee8534fb77e8caf338bac62a396baef/gambar/Screenshot%202026-06-08%20000823.png)

3. Terakhir, masukkan perintah berikut: 'python -m petra058_modul.petra058_utama

Setelah melakukan semua tahap yang ditunjukkan, Terminal akan memanggil semua file pada modul untuk menjalankan sesi penggunaan operasi matriks seperti pada gambar di bawah.
![Tutorial](https://github.com/petrabona/Project-Ujian-Akhir-Semester-Struktur-Data-PSTI-A/blob/d5f738db4ee8534fb77e8caf338bac62a396baef/gambar/Screenshot%202026-06-08%20001056.png)
